# Tarefas:
## Desenvolver as seguintes atividades no formato de fluxograma:

- [x] Exercicio 1: Crie um Fluxograma que
Leia o nome de uma pessoa
Digite “Olá, [tal pessoa]!”

- [x] Exercicio 2: Crie um Fluxograma que
Leia dois números
Calcule a soma
Exiba o resultado da soma

- [x] Exercicio 3: Crie um Fluxograma que 
Leia a idade de uma pessoa
Se a idade for maior ou igual a 18, exiba “Maior de idade”
Caso contrário exiba “Menor de idade”

- [x] Exercicio 4: Crie um Fluxograma que 
Leia um número
Informe se ele é positivo negativo ou zero

- [x] Exercicio 5: Crie um Fluxograma que 
Leia duas notas
Calcule a média
Exiba a média final
